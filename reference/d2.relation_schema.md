# Generate D2 input text to plot relation schemas

Produces text input for D2 to make a diagram of a given relation schema,
usually rendered with SVG.

## Usage

``` r
# S3 method for class 'relation_schema'
d2(x, name = NA_character_, ...)
```

## Arguments

- x:

  a relation schema, as given by
  [`relation_schema`](https://charnelmouse.github.io/autodb/reference/relation_schema.md)
  or
  [`synthesise`](https://charnelmouse.github.io/autodb/reference/synthesise.md).

- name:

  a character scalar, giving the name of the schema, if any.

- ...:

  further arguments passed to or from other methods.

## Value

A scalar character, containing text input for D2.

## Details

Each relation in the schema is presented as a set of rows, one for each
attribute in the relation. These rows do not include information about
the attribute classes.
